\hypertarget{classmesh_1_1_face}{}\doxysection{mesh\+::Face Class Reference}
\label{classmesh_1_1_face}\index{mesh::Face@{mesh::Face}}


Represents a boundary between Cells or between a \mbox{\hyperlink{classmesh_1_1_cell}{Cell}} and the boundary. For memory optimization only 1 copy of each \mbox{\hyperlink{classmesh_1_1_face}{Face}} is created, and a pointer is shared between adjacent Cells. Faces also store a reference to the cells that is boundary of. Further implementation of duplicated faces to increse speed over memory may take place. Only available for 2D cell. If face have only 1 cell when build funtion is called it is considered boundary. \mbox{\hyperlink{classmesh_1_1_cell}{Cell}} 1 can\textquotesingle{}t be null, only \mbox{\hyperlink{classmesh_1_1_cell}{Cell}} 2 can be null in case its boundary.
